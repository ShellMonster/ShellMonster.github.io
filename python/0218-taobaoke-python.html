0218-淘宝客返利机器人源码及搭建教程公布!

一、必备条件：

　　Python开发环境，Python3.x即可，安装教程百度查找即可，比较简单；
　　基于Python环境安装的库，包括requests(主要用于对接接口来返回数据)、pandas(数据的库，主要用来保存用户查询记录，或者说购买的记录存表)、DataFrame(与pandas搭配使用)、re(正则表达式库，用户匹配查找用户所发送消息内淘宝商品信息内容)、quote(主要做编码式链接拼接)、urllib(搭配quote使用)、itchat(微信网页版第三方库，核心)；
　　
　　如需直接获取源码请下滑至最底部，底部也有使用的示例图。

1，安装python3.x环境；
